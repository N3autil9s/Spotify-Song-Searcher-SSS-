# Spotify-Song-Searcher-SSS-
A small code bit that uses a list to and the webbrowser library to find songs.
make sure to enter song titles with " Song name Author ", and a comma after if you want to search up more than one song.
The songs currently in are just a placeholder for others that you might want to put in, there will be an updated version that will search soundcloud and youtube and spotify depending on which platform you are looking for.
